# Mazlíčci E-shop

Testovací e-shop pro mazlíčky v PHP 5.6.  
Data jsou uložená v JSON souborech (`data/*.json`).

## Spuštění

```bash
php -S localhost:8080 -t public
```

http://localhost:8080/shop.php

http://localhost:8080/admin.php

http://localhost:8080/playground.php (develop/testing playground)
